# PRD-025: Business Profile Edit Access

**Status:** Ready for Implementation
**Priority:** P2 - Feature Gap
**Category:** Content Section
**Spec:** [SPEC-025](../../specs/spec-025-business-profile-edit.md)
**Created:** 2026-01-14

---

## Overview

Add ability to edit, view, or update a business profile once it's connected. Currently there's no visible option to manage connected profiles.

## Problem Statement

Users cannot update their business profile information after initial creation. There's no way to correct mistakes or update changing information.

## Goals

1. Enable profile updates
2. Standard account management functionality
3. Reduce user frustration

## User Stories

**US-025.1:** As a CallVault user with a connected business profile, I want to edit my profile information so that I can keep it up to date.

## Requirements

| ID | Requirement | Priority |
|----|-------------|----------|
| FR-001 | Edit button visible for connected profiles | Must Have |
| FR-002 | Clicking opens profile for editing | Must Have |
| FR-003 | Changes can be saved | Must Have |

## Technical Approach

Add edit button to business profile display that opens profile editor modal/panel.

## Acceptance Criteria

- [ ] Edit button visible for connected profiles
- [ ] Clicking opens profile editor
- [ ] Changes can be saved
- [ ] View-only mode available
