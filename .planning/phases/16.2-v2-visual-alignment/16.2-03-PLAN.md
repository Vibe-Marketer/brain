In the `@layer base` section, change the heading rule from:
```css
h1, h2, h3, h4, h5, h6 {
  font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
  font-weight: 600;
}
```

```css
h1, h2, h3, h4, h5, h6 {
```
