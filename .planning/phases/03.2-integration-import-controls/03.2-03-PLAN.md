In SyncTab.tsx, wrap the connectedPlatforms derivation in useMemo:

Current code (around line 64):
```typescript
const connectedIntegrations = integrations.filter((i) => i.connected);
const connectedPlatforms = connectedIntegrations.map((i) => i.platform);
```

Change to:
```typescript
const connectedIntegrations = integrations.filter((i) => i.connected);
const connectedPlatforms = useMemo(
  () => connectedIntegrations.map((i) => i.platform),
  [connectedIntegrations.length] // Only recompute when count changes
);
```

Note: We use `connectedIntegrations.length` as the dependency rather than `connectedIntegrations` because the filter also creates a new array. Using length is sufficient since we only care if integrations were added/removed.
