# SERP Domination Plan Workflow

<objective>
Create a 90-day strategy to dominate search results for your core keywords. Output: Prioritized action plan with weekly milestones and success metrics.
</objective>

<inputs_required>
- Primary keyword targets (5-10)
- Current ranking positions
- Competitor analysis (top 3)
- Available resources (time, budget, team)
- Existing content inventory
</inputs_required>

<process>

## Phase 1: Current State Assessment

**Audit your baseline:**

| Keyword | Current Rank | Target Rank | Competition | Content Exists? |
|---------|--------------|-------------|-------------|-----------------|
| [kw1] | #45 | #5 | High | Yes - needs update |
| [kw2] | - | #10 | Medium | No |
| [kw3] | #12 | #3 | Low | Yes - needs expansion |

**Competitive gap analysis:**

For each target keyword, document:
- Who ranks #1-3?
- What's their content length/depth?
- How many backlinks does top result have?
- What content format do they use?
- What are they missing that you can add?

## Phase 3: Days 1-30 Action Plan

### Week 1: Technical Foundation

| Day | Task | Deliverable |
|-----|------|-------------|
| 1-2 | Site speed audit | PageSpeed report + fixes |
| 3-4 | Crawlability check | Fix crawl errors in GSC |
| 5 | Mobile usability | Fix mobile issues |
| 6-7 | Core Web Vitals | LCP, FID, CLS optimization |

**Technical checklist:**
- [ ] Page speed: LCP < 2.5s, FID < 100ms, CLS < 0.1
- [ ] Mobile-friendly test passes
- [ ] XML sitemap submitted
- [ ] Robots.txt properly configured
- [ ] HTTPS across all pages
- [ ] No duplicate content issues
- [ ] Clean URL structure
- [ ] Proper canonical tags

### Week 2: On-Page Quick Wins

| Day | Task | Deliverable |
|-----|------|-------------|
| 8-9 | Title tag optimization | All pages optimized |
| 10-11 | Meta description optimization | All pages optimized |
| 12-13 | Header structure (H1, H2, H3) | Proper hierarchy |
| 14 | Image optimization | Alt tags, compression |

**On-page checklist per page:**
- [ ] Title: <60 chars, keyword near front
- [ ] Meta: <160 chars, compelling, keyword included
- [ ] H1: One per page, includes primary keyword
- [ ] H2s: Logical structure, secondary keywords
- [ ] Content: Keyword naturally included
- [ ] Images: Descriptive alt text, compressed

### Week 3: Quick Win Content

Target pages ranking 11-20 for priority keywords:

| Page | Current Rank | Optimization Actions |
|------|--------------|---------------------|
| /page-1 | #15 | Add 500 words, update stats, add FAQ |
| /page-2 | #12 | Improve intro, add images, internal links |
| /page-3 | #18 | Complete rewrite needed |

**Content update process:**
1. Analyze top 3 ranking pages
2. Identify what they have that you don't
3. Add missing sections
4. Update outdated information
5. Improve readability
6. Add multimedia
7. Implement schema

### Week 4: Content Audit

**Categorize all existing content:**

| Category | Action | Timeline |
|----------|--------|----------|
| High performers | Monitor, light optimization | Ongoing |
| Underperformers (potential) | Heavy optimization | Month 2 |
| Outdated content | Update or consolidate | Month 2 |
| Thin content | Expand or remove | Month 2 |
| Duplicate content | Consolidate or canonicalize | Week 4 |

## Phase 4: Days 31-60 Action Plan

### Week 5-6: New Content Creation

Create content for identified gaps:

| Priority | Keyword | Content Type | Target Length | Target Date |
|----------|---------|--------------|---------------|-------------|
| 1 | [kw] | Comprehensive guide | 3,000+ words | Day 35 |
| 2 | [kw] | Comparison post | 2,000+ words | Day 40 |
| 3 | [kw] | How-to tutorial | 2,500+ words | Day 45 |

**Content quality standards:**
- Better than current #1 ranking content
- Original insights or data
- Proper schema markup
- Optimized for featured snippets
- Mobile-friendly formatting
- Fast loading (minimal scripts)

### Week 7: Internal Linking Overhaul

**Build content hub structure:**

```
PILLAR PAGE (main keyword)
    │
    ├── Cluster 1 (sub-topic)
    │   ├── Supporting post A
    │   ├── Supporting post B
    │   └── Supporting post C
    │
    ├── Cluster 2 (sub-topic)
    │   ├── Supporting post D
    │   ├── Supporting post E
    │   └── Supporting post F
    │
    └── [All cluster posts link to pillar]
        [Pillar links to all clusters]
```

**Internal linking audit:**
- [ ] Every blog post has 3-5 internal links
- [ ] Pillar pages linked from homepage/nav
- [ ] Related posts section on all articles
- [ ] Contextual links in body copy
- [ ] No orphan pages

### Week 8: Schema Implementation

**Implement across all content:**

| Page Type | Schema Type | Priority |
|-----------|-------------|----------|
| Blog posts | Article + FAQ | High |
| Product pages | Product | High |
| Service pages | Service + FAQ | High |
| About page | Organization + Person | Medium |
| Reviews | Review | Medium |

## Phase 5: Days 61-90 Action Plan

### Week 9-10: Link Building

**Outreach targets:**

| Strategy | Volume | Expected Links |
|----------|--------|----------------|
| Guest posts | 8 pitches | 2-3 links |
| Resource page outreach | 20 emails | 3-5 links |
| Broken link building | 15 opportunities | 2-4 links |
| HARO responses | 20 responses | 1-3 links |

**Link building priorities:**
1. Relevant sites in your industry
2. Sites linking to competitors but not you
3. Resource pages in your niche
4. Industry publications
5. Podcast/interview opportunities

### Week 11: Digital PR

**Create linkable assets:**
- Original research/survey
- Industry report
- Infographic
- Free tool/calculator

**Promotion:**
- Email to industry journalists
- Social media distribution
- Outreach to sites that covered similar topics
- Submit to relevant communities

### Week 12: Review & Iterate

**Performance review:**

| Keyword | Day 1 Rank | Day 90 Rank | Change | Next Action |
|---------|------------|-------------|--------|-------------|
| [kw1] | #45 | #12 | +33 | Continue optimization |
| [kw2] | - | #35 | New | Build links |
| [kw3] | #12 | #5 | +7 | Maintain |

**Adjustment strategies:**
- Keywords showing improvement: Double down
- Keywords stagnant: Analyze competitors, add content
- Keywords declining: Technical or content issue?

## Phase 6: Ongoing Maintenance

**Monthly tasks:**

| Week | Focus |
|------|-------|
| Week 1 | Content updates (refresh top performers) |
| Week 2 | Technical monitoring (speed, errors) |
| Week 3 | Link building outreach |
| Week 4 | Reporting and strategy adjustment |

**Quarterly reviews:**
- Full ranking report
- Competitor position changes
- Content gap re-analysis
- Link profile growth
- Conversion rate analysis
