# Google Business Profile Posts Creation Workflow

<objective>
Create a batch of optimized Google Business Profile posts for local businesses. Output: 10-20 ready-to-publish posts covering different post types.
</objective>

<inputs_required>
- Business name
- Business type/category
- Services offered
- Location/service area
- Any current offers or promotions
- Target keywords (service + location)
</inputs_required>

<process>

## Phase 1: Competitor Research

```
Go to Google Maps
Search "[SERVICE] [LOCATION]"
Check top 3 competitors' GBP posts:
- What types of posts do they use?
- What topics get engagement?
- How often do they post?
- What's missing that we can do better?
```

## Phase 2: Post Type Strategy

**Mix of post types for variety:**

| Post Type | Frequency | Purpose |
|-----------|-----------|---------|
| What's New / Update | 2-3/week | General visibility |
| Offer | 1-2/week | Drive conversions |
| Event | As needed | Promote events |
| Product | 1/week | Highlight services |

## Phase 4: Image Guidelines

Each post should include an image:

| Post Type | Image Recommendation |
|-----------|---------------------|
| Service | Team in action, equipment, location |
| Offer | Bold text overlay with discount |
| Update | Business exterior, team photo |
| Product | Service result, before/after |

Image specs:
- Minimum 720 x 540 pixels
- JPG or PNG format
- No stock photos (authentic preferred)

## Phase 5: Posting Schedule

**Recommended frequency:** 2-4 posts per week

| Day | Post Type |
|-----|-----------|
| Monday | Update/Service highlight |
| Wednesday | Offer or Product |
| Friday | Update or FAQ |
| Sunday (optional) | Behind the scenes |

## Phase 6: Deliverable

Create document with:
1. 10-20 ready-to-post content blocks
2. Suggested posting schedule
3. Image recommendations for each
4. CTA button recommendations
