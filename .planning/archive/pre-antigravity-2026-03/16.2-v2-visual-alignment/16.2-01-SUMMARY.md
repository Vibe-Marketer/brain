## Decisions Made

- **Radix import pattern:** Used `import { Tabs as TabsPrimitive } from 'radix-ui'` and `import { Slot } from 'radix-ui'` — the v2 project uses the unified `radix-ui` package, not individual `@radix-ui/react-*` packages
- **Slot.Root:** The flat import from `radix-ui` exposes Slot as `Slot.Root` (namespace object), not as a named export directly
- **Pill via CSS only:** Active tab indicator uses `data-[state=active]:after:*` Tailwind utilities — no JS state tracking needed, Radix handles it via data attributes
- **Enforced layout:** `justify-start gap-6` appended after `className` in TabsList so user className cannot override brand-required alignment
