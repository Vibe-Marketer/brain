CallVault integrates with the tools you already use to record calls. Connect an integration once and your calls flow in automatically — no manual uploads, no copy-paste.

## What all integrations share

Regardless of which integration you use to bring calls in, every call in CallVault gets the same treatment:

- **Automatic transcription** — full text transcript with speaker labels
- **AI summary** — overview, action items, and key topics
- **Full-text search** — every word indexed immediately
- **Routing rules** — automatically tagged and organized based on your rules
- **Export and sharing** — same export formats and sharing options for all calls

## Connecting an integration

All integrations are configured in **Settings → Integrations**. Each integration shows its connection status, last sync time, and a disconnect option.
