## What's new

### Workspace routing rules

You can now create routing rules to automatically organize incoming calls into the right workspace and folder — no manual sorting required. Set conditions based on meeting title, participant name, duration, or call source, and define actions like tagging or folder assignment.

See [Routing Rules](/automation/routing-rules) for setup instructions.

### Teams

Added support for Teams — named groups of organization members that can be added to workspaces together. Manage access for your whole sales team or CS team in one place instead of member-by-member.

See [Teams](/collaboration/teams) to get started.

### Bulk export

You can now select multiple calls in a list view and export them all at once. Bulk exports download as a ZIP file containing one document per call. CSV bulk export produces a single file with all calls combined.

See [Export](/transcripts/export) for format options.

### Share link improvements

Share links now support password protection and configurable expiry dates. Existing share links continue to work unchanged.

See [Sharing](/transcripts/sharing) for the updated options.

### AI summary regeneration

If a transcript was re-run or corrected, you can now regenerate the AI summary for any call from the **⋯** menu in the summary panel.
