# Free SEO Tools Reference

Use these tools when you don't have paid Ahrefs/SEMrush access. Prioritize in this order.

---

## Tier 1: Best Free Options

### Ubersuggest (ubersuggest.com)
**Free tier:** 3 searches/day
**Best for:** Keyword research, competitor analysis, site audits

**What you can do free:**
- Keyword ideas with volume, CPC, difficulty
- Competitor traffic overview
- Top pages analysis
- Backlink overview
- Site audit (limited)

**Browser commands:**
```
Go to neilpatel.com/ubersuggest
Enter "[KEYWORD]" or "[DOMAIN]"
Select country/language
Extract: keyword ideas, volume, SD (SEO difficulty), PD (paid difficulty)
```

---

### Google Search Console (search.google.com/search-console)
**Free tier:** Unlimited (for your own sites)
**Best for:** Your own site's performance data

**What you can do:**
- See every keyword you rank for
- Impressions, clicks, CTR, position
- Compare time periods (find declining content)
- Find high-impression low-CTR opportunities
- Technical issues (indexing, mobile, core web vitals)

**This is the most valuable free tool you have for your own site.**

---

### Google Keyword Planner (ads.google.com/keyword-planner)
**Free tier:** Unlimited (need Google Ads account, no spend required)
**Best for:** Keyword volume and ideas

**What you can do:**
- Search volume ranges (not exact without ad spend)
- Keyword ideas from seed keywords
- Competition level (for ads, but correlates with SEO)
- CPC data (indicates commercial value)

---

### Google PageSpeed Insights (pagespeed.web.dev)
**Free tier:** Unlimited
**Best for:** Page speed and Core Web Vitals

**What you can do:**
- Mobile and desktop performance scores
- LCP, FID/INP, CLS metrics
- Specific recommendations for improvement

---

### Google Rich Results Test (search.google.com/test/rich-results)
**Free tier:** Unlimited
**Best for:** Schema markup validation

---

## Tier 2: Useful Free Tools

### AnswerThePublic (answerthepublic.com)
**Free tier:** 3 searches/day
**Best for:** Question-based keywords, PAA research

### AlsoAsked (alsoasked.com)
**Free tier:** 3 searches/day
**Best for:** PAA (People Also Ask) mining

### Keyword Surfer (Chrome extension)
**Free tier:** Unlimited
**Best for:** See search volume directly in Google SERPs

### SEOquake (Chrome extension)
**Free tier:** Unlimited
**Best for:** Quick on-page SEO analysis

### Screaming Frog (screamingfrog.co.uk)
**Free tier:** 500 URLs
**Best for:** Technical site audits

### Ahrefs Webmaster Tools (ahrefs.com/webmaster-tools)
**Free tier:** Unlimited for verified sites
**Best for:** Backlink data for YOUR site only

---

## Tier 3: Free Trials Worth Using

### Semrush
**Free trial:** 7 days full access
**Strategy:** Save for when you need deep competitor research

### Ahrefs
**Free trial:** $7 for 7 days (not free, but cheap)
**Strategy:** Batch all your research into one week

### Moz
**Free trial:** 30 days
**Best for:** Domain authority checking, link research

---

## Recommended Free Stack

**For keyword research:**
1. Ubersuggest (3/day) - primary
2. Google Keyword Planner - backup/validation
3. AnswerThePublic (3/day) - questions
4. Keyword Surfer extension - passive data

**For competitor analysis:**
1. Ubersuggest (3/day) - traffic, keywords, backlinks
2. SimilarWeb free - traffic estimates
3. Manual SERP analysis - see who ranks

**For your own site:**
1. Google Search Console - rankings, CTR, issues
2. Google Analytics - traffic patterns
3. Ahrefs Webmaster Tools - your backlinks
4. PageSpeed Insights - speed

**For content optimization:**
1. Clearscope/Surfer free trials - content scoring
2. Hemingway Editor - readability
3. Schema markup generators - free online tools

## Maximizing Free Searches

**Ubersuggest 3/day strategy:**
- Day 1: Your domain overview + top competitor
- Day 2: Main keyword + secondary keyword
- Day 3: Third competitor + content gap keyword

**Batch your research:**
When you use a free search, extract EVERYTHING:
- Screenshot all data
- Export if possible
- Document in a file

**Use free trials strategically:**
Don't start a trial until you have a specific research project. Then extract all the data you need in that window.

When you ask me to do research, I'll use tools in this order:
1. Google tools (unlimited, always available)
2. Ubersuggest (3/day limit)
3. Free extensions and manual analysis
4. Paid tools only if you have access
